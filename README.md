# *FaceDetetor*

准实时CPU深度学习人脸检测

## 如何使用

### 直接使用

运行Faced-Main.py

### 命令行使用

```bash
# 检测图片保存输出
$ python faced --input imgs/demo.png --save
```

```bash
# 检测图片保存输出
$ python faced --input imgs/demo.mp4 --save
```

```bash
# 摄像头在线检测
$ python faced --input webcam
```

### bug

🚧 视频检测时，视频播放完后会自动关闭所有窗口 🚧
